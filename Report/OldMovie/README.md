# 实验报告

### 在vscode中拆分终端分别进入后端和前端

进入后端

```
cd .\CryptoDrive\
cd .\backend\
npm install
npm run
```

进入后端

```
cd .\CryptoDrive\
cd .\frontend\
npm install
npm start
```
